### 账号
账号不做赘述/可以自己找渠道注册or购买

### 代理禁止访问
所有禁止访问的请清除cookie重新访问

或浏览器地址栏输入javascript:window.localStorage.removeItem(Object.keys(window.localStorage).find(i=>i.startsWith('@@auth0spajs')))回车
